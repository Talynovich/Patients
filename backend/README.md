# APIPatients

## Auth

| Метод | Эндпоинт | Описание                              | Доступ |
| :--- | :--- |:--------------------------------------| :--- |
| `POST` | `/auth/register` | Регистрация нового пользователя       | Публичный |
| `POST` | `/auth/login` | Авторизация и получение токенов       | Публичный |
| `POST` | `/auth/admin-setup` | Создание Admin пользователя           | Публичный |


## Patients

| Метод | Эндпоинт | Описание                              | Доступ |
| :--- | :--- |:--------------------------------------| :--- |
| `GET` | `/patients` | Получение списка всех пациентов       | Доктор / Админ |
| `GET` | `/patients/:id` | Получение данных конкретного пациента | Доктор / Админ |
| `POST` | `/patients` | Добавление нового пациента            | Доктор |

## Users

| Метод  | Эндпоинт | Описание                              | Доступ        |
|:-------| :--- |:--------------------------------------|:--------------|
| `POST` | `/users/doctors` | Регистрация нового врача       | Админ         |
| `GET`  | `/users/doctors` | Посмотреть список всех врачей |  Админ        |

## Appointment

| Метод  | Эндпоинт | Описание                              | Доступ        |
|:-------| :--- |:--------------------------------------|:--------------|
| `POST` | `/appointments` | Создание записи на прием       | Доктор         |
| `GET`  | `/appointments/my` | Список приемов текущего врача |  Доктор        |
